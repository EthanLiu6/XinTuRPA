Agent Skill：通用营销需求-达人搜索（目标人群+核心关键词）提取

一、核心定位

全领域通用，从任意行业/品类营销需求文档中，**分层提取核心/圈外目标人群画像**、**分维度生成3-5个核心/圈外达人搜索核心关键词**，直接支撑达人库精准检索与初筛，聚焦搜索核心需求，无行业、场景限制。

二、核心能力

2.1 目标人群画像提取（核心/圈外分层提取）

自动识别文档中人群相关信息，按**核心人群、圈外人群**两层划分，每层均固定3个维度整合输出；优先抓取文档明确表述，无明确表述则从营销目标/推广需求提炼隐含信息，缺项标注「无明确信息」。

通用提取维度（核心/圈外人群均适用）

1. 基础属性：性别、年龄区间、地域范围、职业/身份；

2. 行为偏好：兴趣领域、消费/使用习惯、触媒习惯、品类关注方向；

3. 核心标签：文档定义的人群专属特征标签。

人群分层定义（全领域通用）

- - 核心人群：产品/项目的**核心受众/忠实用户/刚需群体**，是营销首要触达、转化的核心对象；

- - 圈外人群：产品/项目的**潜在受众/泛兴趣用户/拓圈目标群体**，是营销拉新、破圈的触达对象。

2.2 核心关键词生成（核心/圈外分维度生成）

基于需求文档4大核心要素，按**核心人群适配、圈外人群适配**两个维度组合生成；关键词以**名词/名词短语**为主，检索性强、无重复、高度贴合达人创作标签，可直接复制用于达人库搜索，**整体数量严格控制在3-5个（可跨维度复用核心词）**。

通用生成要素（核心/圈外关键词均从以下要素组合）

1. 品类/主体：品牌名/产品名/项目名、核心品类；

2. 营销场景：核心营销动作/节点；

3. 人群标签：核心/圈外人群的核心特征；

4. 内容形式：达人需创作的内容类型。

三、执行规则（标准化，全行业/文档通用）

3.1 人群画像提取规则

1. 优先抓取文档中「目标人群」「受众特征」「核心用户/潜在用户」等明确标注板块的信息，无明确板块则从「项目目的」「推广需求」「拉新/破圈策略」中提炼分层信息；

2. 信息去模糊化，剔除“大众人群”“广泛用户”等无意义描述，保留具体、可匹配的精准信息；

3. 某一维度无明确信息时，直接标注「无明确信息」，不随意拓展；

4. 若文档未明确划分圈外人群，仅提取核心人群画像，标注「圈外人群无明确信息」。

3.2 关键词生成规则

1. 数量固定：整体3-5个，核心/圈外维度可复用1-2个核心通用词，再补充差异化关键词；

2. 强匹配性：所有关键词均来自文档核心信息，核心关键词贴合核心人群需求，圈外关键词贴合拓圈/拉新需求，不添加无关拓展词；

3. 唯一性：无重复/近似关键词，同一维度择一核心表述；

4. 通用性：避免行业生僻词，选用达人库常见创作/标签关键词。

四、投放目标

**投放目标**：基于营销需求文档，精准匹配并提取核心投放方向，包括但不限于**核心人群精准触达**（聚焦核心受众提升转化效率）、**圈外人群拓圈拉新**（挖掘潜在用户扩大受众范围）、**内容场景深度匹配**（适配营销场景与内容形式提升内容效果）、**达人检索效率优化**（快速筛选符合需求达人支撑活动落地），确保目标人群画像与核心关键词完全服务于实际投放需求，无明确投放目标时标注「无明确投放目标」。

五、适配范围

1. 文档类型：营销需求Brief、采购需求表、品牌推广方案、活动需求书、品类营销规划等所有营销相关文档；

2. 行业领域：美妆护肤、食品饮料、数码3C、户外露营、母婴亲子、服饰鞋包、家居日用、游戏娱乐、文旅出行、汽车用品等全行业全品类；

3. 营销场景：新品上市、节日大促、换季种草、品牌曝光、用户拉新/留存、直播大事件、内容测评、场景体验、品牌破圈等所有营销场景。

七、多行业示例

示例1：美妆护肤-某新品粉底液营销需求

【目标人群画像（核心/圈外分层）】

核心人群

1. 基础属性：女性为主，20-35岁，一线/新一线城市，职场白领/精致女性；

2. 行为偏好：长期关注美妆护肤，有固定底妆使用习惯，活跃于小红书/抖音，热衷专业底妆测评内容；

3. 核心标签：底妆深度爱好者、美妆精致党、新锐白领。

圈外人群

1. 基础属性：女性为主，18-40岁，新一线/二三线城市，学生/轻美妆用户；

2. 行为偏好：偶尔关注美妆内容，底妆产品更换频率低，易被高颜值/高性价比内容吸引，活跃于抖音/视频号；

3. 核心标签：轻美妆爱好者、性价比追求者、美妆新手。

【达人搜索核心关键词（核心/圈外分维度）】

共5个：粉底液、美妆护肤、底妆测评、轻美妆、性价比底妆

核心人群适配关键词：底妆测评、粉底液

圈外人群适配关键词：轻美妆、性价比底妆

通用核心词：美妆护肤

示例2：户外露营-某露营装备品牌秋季推广需求

【目标人群画像（核心/圈外分层）】

核心人群

1. 基础属性：男女均有，25-45岁，一线/新一线城市，职场白领/自由职业者，有露营经验；

2. 行为偏好：热衷户外露营，关注专业露营装备，有固定露营社群，活跃于小红书/抖音，喜欢看装备测评/露营攻略；

3. 核心标签：资深露营玩家、户外爱好者、露营装备发烧友。

圈外人群

1. 基础属性：男女均有，20-40岁，新一线/二三线城市，职场白领/宝妈，无露营经验或轻度露营体验；

2. 行为偏好：关注户外休闲生活，易被亲子/闺蜜露营场景吸引，注重露营氛围感，活跃于抖音/视频号；

3. 核心标签：户外休闲爱好者、亲子露营潜在用户、氛围感追求者。

【达人搜索核心关键词（核心/圈外分维度）】

共4个：露营装备、户外露营、露营装备测评、亲子露营

核心人群适配关键词：露营装备测评、露营装备

圈外人群适配关键词：亲子露营

通用核心词：户外露营

示例3：母婴亲子-某新款儿童奶粉新品上市需求

【目标人群画像（核心/圈外分层）】

核心人群

1. 基础属性：女性（宝妈）为主，28-40岁，全国范围，0-3岁宝宝家长；

2. 行为偏好：关注母婴育儿专业内容，重视奶粉配方/安全性，活跃于小红书/母婴社群，喜欢看专业奶粉测评；

3. 核心标签：母婴刚需宝妈、0-3岁育儿人群、专业育儿爱好者。

圈外人群

1. 基础属性：女性（备孕/待产宝妈）为主，25-35岁，全国范围，备孕/待产人群；

2. 行为偏好：初步关注母婴育儿内容，注重奶粉口碑/性价比，易被育儿干货内容吸引，活跃于抖音/小红书；

3. 核心标签：备孕/待产宝妈、奶粉潜在用户、新手育儿人群。

【达人搜索核心关键词（核心/圈外分维度）】

共4个：儿童奶粉、母婴育儿、奶粉测评、新手育儿

核心人群适配关键词：奶粉测评

圈外人群适配关键词：新手育儿

通用核心词：儿童奶粉、母婴育儿
