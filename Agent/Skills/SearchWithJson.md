Agent Skill：营销需求（投放/关键词/品类等）JSON格式转换

一、核心定位

聚焦营销需求文档，提取投放目标、关键词列表、品类、用户划分、其他信息5类核心内容，按规范格式转换为JSON，输出简洁、精准、可直接调用，适配全行业营销需求，无场景限制。

二、核心能力

2.1 核心内容提取

自动识别营销需求文档中指定5类核心信息，优先抓取明确标注内容，无明确表述则基于文档核心信息提炼，缺项标注null，不随意拓展无关内容。

- 投放目标：提取文档中核心投放方向（如精准触达、拓圈拉新等），多个目标用数组呈现；

- 关键词列表：提取达人搜索核心关键词，保留所有核心/圈外适配词、通用核心词，去重后按数组呈现；

- 品类：提取文档中产品/项目所属核心品类，明确单一或多个核心品类；

- 用户划分：按核心人群、圈外人群分层提取关键信息（基础属性、核心标签），简洁呈现分层逻辑；

- 其他信息：提取文档中未涵盖在以上4类、且对JSON调用有价值的信息（如营销场景、核心诉求等）。

2.2 JSON格式转换

按固定JSON结构整理提取内容，格式规范、层级清晰，无冗余字段，支持直接复制调用，确保数据类型统一（字符串、数组对应匹配）。

三、执行规则

1. 信息提取：优先抓取文档中“投放目标”“关键词”“品类”“目标人群”等明确标注板块，无明确板块则从核心内容提炼；

2. 格式规范：严格遵循固定JSON结构，字段名称不修改，缺项统一标注null，不遗漏指定5类内容；

3. 精准性：提取内容与文档高度匹配，不添加无关信息，关键词去重、用户划分贴合原文分层；

4. 简洁性：JSON输出无冗余字段，用户划分、投放目标等内容简洁提炼，不冗长。

四、适配范围

1. 文档类型：营销需求Brief、品牌推广方案、活动需求书等所有含指定5类信息的营销相关文档；

2. 行业领域：美妆护肤、户外露营、母婴亲子等全行业全品类（与前序达人搜索技能适配）。

五、标准化JSON输出格式（固定不变）

{
  "投放目标": [String/Array, 缺项填null],
  "关键词列表": [String/Array, 去重后呈现, 不为空],
  "品类": [String/Array, 核心品类, 缺项填null],
  "用户划分": {
    "核心人群": {
      "基础属性": String,
      "核心标签": String
    },
    "圈外人群": {
      "基础属性": String,
      "核心标签": String,
      "无明确信息则填null"
    }
  },
  "其他信息": String/Array, 无则填null
}

六、示例（适配前序达人搜索技能案例）

    },
