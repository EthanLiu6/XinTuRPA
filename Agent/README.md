# 星图达人需求分析系统

基于大模型的智能需求分析系统，用于处理客户营销需求文档，提取目标人群画像和关键词，生成标准化达人搜索参数。

## 功能特性

- 📄 **文档上传**: 支持 txt, doc, docx, pdf, md 等格式
- 🤖 **智能分析**: 使用 Agent Skill 自动提取人群画像
- 🔑 **关键词生成**: 核心人群/圈外人群分层关键词
- 📋 **JSON 输出**: 生成标准化的搜索参数

## 安装

```bash
npm install
```

## 启动

```bash
# 终端1：启动后端服务
npm run server

# 终端2：启动前端开发服务器
npm run dev
```

访问 http://localhost:5174

## 使用流程

1. **上传需求文档**: 拖拽或点击上传客户需求文档
2. **配置 API Key**: 输入 OpenAI API Key
3. **开始分析**: 点击"开始分析"按钮
4. **查看结果**: 查看人群画像、关键词、JSON 参数
5. **导出结果**: 复制或导出分析结果

## Agent Skills

### DemandMsg Skill
从营销需求文档中提取：
- 核心人群画像（基础属性、行为偏好、核心标签）
- 圈外人群画像
- 核心关键词列表

### SearchWithJson Skill
将分析结果转换为标准 JSON 格式：
```json
{
  "投放目标": ["核心人群精准触达", "圈外人群拓圈拉新"],
  "关键词列表": ["防晒", "美妆护肤", "敏感肌"],
  "品类": "美妆护肤-防晒",
  "用户划分": {
    "核心人群": { ... },
    "圈外人群": { ... }
  },
  "其他信息": "..."
}
```

## 与达人搜索系统对接

分析完成后，将生成的关键词列表复制到 [XinTuRPA](../) 系统中进行达人搜索。
