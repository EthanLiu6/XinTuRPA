# XinTuRPA（星图达人抓取系统）

Vue 3 + Vite 可视化界面 + 命令行模式，自动抓取抖音星图平台达人数据。

## 安装

```bash
npm install
```

## 使用方式

### 方式一：Vue开发模式（推荐）

```bash
# 终端1：启动API服务器
npm run server

# 终端2：启动Vue开发服务器
npm run dev
```

访问 http://localhost:5173

### 方式二：生产模式

```bash
npm run build      # 构建Vue应用
npm run server     # 启动服务器
```

访问 http://localhost:3000

### 方式三：命令行模式

```bash
npm start
```

登录后自动抓取，导出到 `output/` 目录。

## 操作流程

1. 点击「启动浏览器」按钮
2. 在弹出的浏览器中完成抖音星图登录
3. 点击「检查登录」确认登录状态
4. 点击「确认登录」进入抓取准备
5. 添加关键词（支持快捷添加，设置页数）
6. 点击「开始抓取」执行RPA
7. 查看数据看板和结果表格
8. 下载CSV/JSON文件

## 功能特性

### Web界面功能
- Vue 3 现代化组件化开发
- 实时状态显示和进度跟踪
- 关键词管理（添加/删除/页数设置）
- 快捷关键词按钮
- 数据看板统计（总达人/关键词数/有报价/有橱窗）
- 结果表格展示（搜索/排序/分页）
- 文件下载

## 配置

编辑 `config.json`：

```json
{
  "browser": "msedge",
  "outputDir": "./output"
}
```

## 输出格式

### CSV 字段（25列）

| 字段 | 说明 |
|------|------|
| 昵称 | 达人昵称 |
| 达人ID | 达人唯一标识 |
| 认证类型 | 蓝V/黄V/个人/无 |
| 粉丝数 | 格式化显示（万/亿） |
| 获赞数 | 总获赞数 |
| 地区 | 省份+城市 |
| 性别 | 男/女/未知 |
| 互动率 | 30天互动率 |
| 平均播放 | 30天平均播放量 |
| 橱窗 | 有/无 |
| 电商等级 | L1-L6 |
| GMV 30天 | 30天成交额区间 |
| 客单价 | 平均客单价区间 |
| 1-20秒报价 | 1-20秒视频报价 |
| 20-60秒报价 | 21-60秒视频报价 |
| 60秒+报价 | 60秒以上视频报价 |
| CPM参考价 | CPM参考价格 |
| 合作参考价 | 30天合作参考价 |
| 星图指数 | 星图综合指数 |
| 转化指数 | 转化能力指数 |
| 内容标签 | 内容主题标签 |
| 最近播放/点赞/分享/评论 | 最近作品数据 |

## 运行失败排查

- 浏览器无法启动：确保已安装 Playwright 浏览器驱动
- 登录超时：检查网络连接，或调整 `loginTimeoutMs`
- 截图保存：自动保存在 `output/screenshots/` 目录
